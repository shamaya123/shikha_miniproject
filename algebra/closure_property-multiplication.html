Multiplicative Identity



a * 1 = a

Example: if a = 3 we have 3 * 1 = 3



Multiplicative Inverse



a * (1/a) = 1

Example: if a = 3, we have 3 * (1/3) = 1, please note that a must not be zero.

(Multiplication times 0)

a * 0 = 0

Example: if a = 3 we have 3 * 0 = 0



Associative of Multiplication



(a * b) * c = a * (b * c)

Example:If a = 2, b= 3 and c = 4, we have: (2*3)*4 = 2*(3*4)



Commutative of Multiplication



a * b = b * a

Example:If a = 2 and b = 3, we have: 2 * 3 = 3 * 2



Distributive Law



a(b + c) = ab + ac

Example:If a = 2, b= 3 and c = 4, we have 2 * (3+4) = 2*3 + 2* 4 = 14



Definition of Division



a / b = a(1/b)

Example:If a = 2 and b = 3, we have 2/3 = 2(1/3)
